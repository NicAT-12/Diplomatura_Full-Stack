# Diplomatura Full Stack

¡Bienvenido a mi repositorio de la Diplomatura Full Stack!

En este espacio iré documentando mi proceso de aprendizaje y crecimiento como desarrollador, incorporando proyectos, prácticas y actividades realizadas a lo largo de la cursada.

## Contenido

Aquí encontrarás trabajos relacionados con:

* HTML y CSS
* JavaScript
* React
* Backend
* Bases de datos
* APIs
* Buenas prácticas de desarrollo
* Proyectos integradores

Cada módulo contará con sus respectivas actividades y proyectos, organizados en carpetas para facilitar su navegación y seguimiento.

## Objetivo

El objetivo de este repositorio es registrar mi evolución como desarrollador Full Stack, consolidar conocimientos mediante proyectos prácticos y construir un portfolio técnico que refleje las habilidades adquiridas durante la diplomatura.
