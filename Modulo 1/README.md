# Prácticas del Módulo 1 - HTML, CSS, JavaScript y React

## Descripción

Este repositorio reúne las actividades realizadas durante el Módulo 1, enfocadas en los fundamentos del desarrollo web utilizando HTML, CSS, JavaScript y React. A lo largo de las tareas se trabajaron conceptos de maquetación web, estilos, manipulación del DOM, programación orientada a objetos, asincronismo y creación de componentes reutilizables en React.

---

## Tarea 1 - HTML y CSS Básico

Desarrollo de una página web utilizando HTML semántico y CSS externo.

### Conceptos trabajados

* Estructura semántica con `header`, `main`, `article` y `footer`.
* Navegación mediante listas y enlaces.
* Uso de imágenes, listas y formularios.
* Aplicación de estilos mediante CSS externo.
* Variables CSS, selectores de clase e ID.
* Efectos visuales con `hover`.
* Buenas prácticas de accesibilidad y organización del código.

---

## Tarea 2 - JavaScript Esencial

Creación de un formulario de registro con validación de edad mediante JavaScript.

### Conceptos trabajados

* Selección de elementos con `querySelector`.
* Manejo de eventos de formularios.
* Uso de `preventDefault()`.
* Obtención y validación de datos ingresados por el usuario.
* Operadores de comparación y estructuras condicionales.
* Manipulación dinámica del DOM.
* Aplicación y eliminación de clases CSS con `classList`.

---

## Tarea 3 - JavaScript Avanzado

Implementación de un gestor de tareas utilizando Programación Orientada a Objetos y asincronismo.

### Conceptos trabajados

* Creación y utilización de clases.
* Métodos y propiedades de objetos.
* Gestión de colecciones mediante arrays.
* Uso de métodos `forEach`, `find`, `filter` y `map`.
* Creación y consumo de Promesas.
* Simulación de operaciones asíncronas con `setTimeout`.
* Flujo asíncrono mediante `async/await`.
* Organización y administración de tareas mediante lógica orientada a objetos.

---

## Tarea 4 - React Inicial

Desarrollo de una aplicación React utilizando Vite y componentes reutilizables.

### Conceptos trabajados

* Configuración de proyectos React con Vite.
* Creación de componentes funcionales.
* Uso de props para reutilización de componentes.
* Renderizado dinámico de información.
* Estructuración de aplicaciones React.
* Estilización mediante CSS.
* Diseño responsive para diferentes tamaños de pantalla.

## Objetivos Alcanzados

* Comprender la estructura básica de una página web.
* Aplicar estilos utilizando CSS externo.
* Manipular el DOM mediante JavaScript.
* Implementar validaciones e interacción con formularios.
