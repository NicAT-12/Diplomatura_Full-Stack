# Registro al Evento

## Descripción

Registro al Evento es una aplicación web desarrollada con HTML5, CSS3 y JavaScript Vanilla como práctica de manipulación del DOM, manejo de eventos y validación de formularios.

La aplicación permite ingresar un nombre y una edad para validar si el usuario puede acceder a un evento según el requisito de ser mayor de edad.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## Características

* Formulario de registro con nombre y edad.
* Validación de campos obligatorios.
* Conversión de datos mediante `Number()`.
* Validación de mayoría de edad utilizando operadores de comparación.
* Manejo de eventos con `addEventListener()`.
* Prevención del comportamiento por defecto con `event.preventDefault()`.
* Manipulación dinámica del DOM mediante `textContent`.
* Aplicación dinámica de estilos con `classList.add()` y `classList.remove()`.
* Interfaz visual moderna y responsive.
